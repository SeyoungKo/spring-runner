### springn boot

특징
1. 단순히 실행되어 스프링 애플리케이션을 쉽게 만들 수 있다.
2. spring 구성이 거의 필요하지 않다.
3. java-jar로 실행하는 java 애플리케이션을 만들 수 있다.

목표
1. spring 개발을 빠르고 광범위하게 적용
2. 설정을 바꿀 수 있다.
3. 대규모 프로젝트에 공통적인 기능제공
4. XML 구성 요구사항이 없음
